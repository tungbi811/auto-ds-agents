# Data Information

## Data Type
### ID type
- PassengerId

### Numerical type
- Age
- SibSp (Number of siblings/spouses aboard)
- Parch (Number of parents/children aboard)
- Fare

### Categorical type
- Survived (0 = No, 1 = Yes)
- Pclass (Passenger class: 1st, 2nd, 3rd)
- Name
- Sex
- Ticket
- Cabin
- Embarked (Port of embarkation: C = Cherbourg, Q = Queenstown, S = Southampton)

### Datetime type
- None

## Detailed data description
The dataset consists of information about passengers aboard the Titanic, with a total of 12 features in the training data and 11 in the test data. The target variable, 'Survived', indicates whether a passenger survived (1) or did not survive (0). The dataset includes various demographic and ticketing details, such as:
- **Age**: There are missing values in this feature, indicating some passengers' ages are not recorded.
- **SibSp and Parch**: These features provide insights into family relationships aboard the ship. 
- **Fare**: Represents the ticket price, which varies significantly among passengers.
- **Embarked**: The port of embarkation shows where passengers boarded the Titanic.

The dataset may reveal patterns regarding survival rates based on demographics, such as gender, age, and social class.

## Target Variable
- Survived: This binary variable indicates whether a passenger survived (1) or did not survive (0). It is the main focus of the analysis to understand factors contributing to survival.
